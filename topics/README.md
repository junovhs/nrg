# topics/

The core of the repo. One folder per research topic, named with a short
`lowercase-with-hyphens` slug.

Start a new topic by copying [`_TEMPLATE/`](_TEMPLATE/) to `topics/<slug>/`, then
add a row to [`../INDEX.md`](../INDEX.md).

Each topic folder contains:
- `README.md` — overview, status, key questions, links.
- `findings.md` — what we've actually learned, with sources.
- `working/` — raw notes, drafts, scratch.
