# sources/

Shared reference material usable across topics: papers, PDFs, datasets, and link
stubs.

- Save files with descriptive `lowercase-with-hyphens` names.
- For web pages you can't save, create a small `.md` stub:

```md
# <Title>
- URL: <link>
- Accessed: YYYY-MM-DD
- Summary: <one line>
```

Reference sources from topics by linking to the file here.
